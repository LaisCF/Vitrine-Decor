# Vitrine-Decor
O Portal Vitrine Decor é uma landing page para serviços de decoração de eventos. Feito com HTML5, CSS3 e JS Vanilla, foca em performance e responsividade. Destaca-se pela galeria inteligente com CSS Grid/Flexbox, imagens sem distorção e filtros em tempo real. Inclui efeito Lightbox e contato direto via API do WhatsApp.
